# Seven Sons AI Assistant

A modern AI conversation platform with 7 unique AI characters

## Features

- **7 Unique AI Characters**: Li Bai, Sun Wukong, Zhuge Liang, Lin Daiyu, Mo Zi, Zhuang Zi, Lu Ban
- **Real-time Chat**: Interactive conversations with AI characters
- **User Authentication**: Secure login and registration
- **Modern UI**: Built with Next.js and Tailwind CSS

## Tech Stack

- **Frontend**: Next.js 14 + TypeScript + Tailwind CSS
- **Backend**: Supabase (Database + Auth + Storage)
- **Payment**: Stripe integration
- **Deployment**: Vercel

## Quick Start

```bash
npm install
npm run dev
```

## Environment Variables

Set up your `.env.local` file with:

```env
NEXT_PUBLIC_SUPABASE_URLýyour_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUP@BASE_SERVICE_ROLE_KEY=your_service_role_key
```
